# Navid-rji Tap

## How do I install these formulae?

`brew install navid-rji/tap/<formula>`

Or `brew tap navid-rji/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "navid-rji/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
